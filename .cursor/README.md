# Cursor Rules

This directory contains rules for the Cursor AI assistant to follow when helping with this project.

## How It Works

Each `.mdc` file contains:

1. A YAML frontmatter section with:

   - `description`: Brief description of the rule file
   - `globs`: File patterns that these rules apply to

2. Markdown-formatted rules and guidelines

## Updating Rules

When updating rules:

1. Choose the appropriate domain file
2. Keep the format consistent
3. Be specific and clear in rule descriptions
4. Update the meeting notes to document significant changes
